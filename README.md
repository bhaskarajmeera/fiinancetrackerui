## Installation

### 1. Clone the repository

> ```
> git clone https://github.com/bhaskarajmeera/ftfrontend
> cd ft-frontend
> ```

### 2. Install dependancies

> ```
> npm install
> ```
>
> yarn
>
> ```
> yarn dev
> ```

### 3. Start Server

> ```
> node app.js
> ```

### 4. Open your Browser

> ```
> http://localhost:8000
> ```
